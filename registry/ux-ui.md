# User Experience and User Interface

Usability engineering, visual and interaction design, automotive HMI ergonomics,
and design systems. Covers ISO 9241 series usability standards, automotive
driver distraction assessment methods, and platform design guidelines from
Google, Apple, and Microsoft.

**Contents:** [Standard](#standard) | [Publication](#publication)

## Standard

- [ISO-14915] Software ergonomics for multimedia user interfaces

  Multi-part standard for designing multimedia user interfaces. Covers design
  principles, navigation, media selection and combination, and temporal
  behaviour of multimedia content in interactive systems.

  Document: ISO 14915 (series, 2003)\
  URL: <https://www.iso.org/standard/28581.html>\
  Label: Standard\
  Keywords: multimedia, user interfaces, navigation, media design, interaction
  design

- [ISO-15005] Road vehicles — Ergonomic aspects of transport information and
  control systems — Dialogue management principles and compliance procedures

  Defines principles for managing driver-vehicle dialogue to minimise
  distraction. Covers interruptibility, resumability, and prioritisation of
  information presentation while driving. Referenced by automotive HMI
  compliance processes.

  Document: ISO 15005:2017\
  URL: <https://www.iso.org/standard/55716.html>\
  Label: Standard\
  Keywords: automotive HMI, driver distraction, dialogue management, in-vehicle
  systems, ergonomics

- [ISO-15008] Road vehicles — Ergonomic aspects of transport information and
  control systems — Specifications and test procedures for in-vehicle visual
  presentation

  Specifies minimum legibility requirements for in-vehicle displays including
  character height, contrast ratio, colour, and viewing angle. Used for type
  approval of instrument clusters, head-up displays, and infotainment screens.

  Document: ISO 15008:2017\
  URL: <https://www.iso.org/standard/62805.html>\
  Label: Standard\
  Keywords: in-vehicle displays, legibility, contrast ratio, HUD, instrument
  cluster, automotive HMI

- [ISO-16673] Road vehicles — Ergonomic aspects of transport information and
  control systems — Occlusion method to assess visual demand due to the use of
  in-vehicle systems

  Defines the occlusion test method for measuring visual demand of in-vehicle
  tasks. Subjects perform tasks while vision is intermittently blocked,
  measuring total shutter open time (TSOT) as a proxy for glance demand.

  Document: ISO 16673:2017\
  URL: <https://www.iso.org/standard/62974.html>\
  Label: Standard\
  Keywords: occlusion method, visual demand, TSOT, driver distraction, glance
  demand, automotive HMI

- [ISO-17287] Road vehicles — Ergonomic aspects of transport information and
  control systems — Procedure for assessing suitability for use while driving

  Defines a checklist-based procedure for assessing whether an in-vehicle system
  is suitable for use while driving. Covers visual, manual, and cognitive demand
  evaluation with pass/fail criteria.

  Document: ISO 17287:2003\
  URL: <https://www.iso.org/standard/30597.html>\
  Label: Standard\
  Keywords: suitability assessment, driver distraction, cognitive demand,
  in-vehicle systems, automotive HMI

- [ISO-20282] Usability of consumer products and products for public use

  Multi-part standard for assessing the ease of operation of everyday products.
  Covers summative test methods and rating scales for walk-up-and- use products
  from ticket machines to household appliances.

  Document: ISO/IEC 20282 (series, 2006)\
  URL: <https://www.iso.org/standard/34122.html>\
  Label: Standard\
  Keywords: consumer usability, product usability, summative testing,
  walk-up-and-use, public use

- [ISO-25062] Systems and software engineering — Systems and software Quality
  Requirements and Evaluation (SQuaRE) — Common Industry Format (CIF) for
  usability test reports

  Defines a standardised format for reporting usability test results. Enables
  consistent comparison of usability data across products, projects, and
  organisations. Required by some regulatory submissions.

  Document: ISO/IEC 25062:2006\
  URL: <https://www.iso.org/standard/43046.html>\
  Label: Standard\
  Keywords: usability testing, CIF, test reports, SQuaRE, usability data,
  reporting format

- [ISO-9241-11] Ergonomics of human-system interaction — Usability: Definitions
  and concepts

  Defines usability as effectiveness, efficiency, and satisfaction in a
  specified context of use. The foundational definition referenced by all
  usability standards, test methods, and regulatory requirements worldwide.

  Document: ISO 9241-11:2018\
  URL: <https://www.iso.org/standard/63500.html>\
  Label: Standard\
  Keywords: usability, effectiveness, efficiency, satisfaction, context of use,
  human-system interaction

- [ISO-9241-125] Ergonomics of human-system interaction — Guidance on visual
  presentation of information

  Provides guidance on visual design of displayed information including
  typography, colour coding, icons, layout, and grouping. Applicable to any
  interactive system with visual output from control panels to mobile apps.

  Document: ISO 9241-125:2017\
  URL: <https://www.iso.org/standard/64839.html>\
  Label: Standard\
  Keywords: visual design, typography, colour coding, icons, layout, information
  presentation

- [ISO-9241-161] Ergonomics of human-system interaction — Guidance on visual
  user-interface elements

  Provides guidance on the design and use of standard visual UI elements
  including buttons, menus, toolbars, lists, trees, tables, and dialogs. Covers
  appearance, behaviour, and interaction patterns for GUI components.

  Document: ISO 9241-161:2016\
  URL: <https://www.iso.org/standard/60476.html>\
  Label: Standard\
  Keywords: UI elements, buttons, menus, dialogs, GUI components, interaction
  patterns

- [ISO-9241-220] Ergonomics of human-system interaction — Processes for
  enabling, executing and assessing human-centred design within organizations

  Defines organisational processes for implementing human-centred design. Covers
  governance, planning, resourcing, and maturity assessment of HCD capabilities
  across projects and product lines.

  Document: ISO 9241-220:2019\
  URL: <https://www.iso.org/standard/63462.html>\
  Label: Standard\
  Keywords: human-centred design, HCD, organisational processes, maturity
  assessment, UX governance

## Publication

- [Apple-HIG] Apple Human Interface Guidelines

  Apple's design guidelines for iOS, macOS, watchOS, tvOS, and visionOS. Defines
  interaction patterns, navigation, controls, visual design, and
  platform-specific conventions. Required reference for CarPlay UI development.

  Document: Apple HIG (continuously updated)\
  URL: <https://developer.apple.com/design/human-interface-guidelines/>\
  Label: Publication\
  Keywords: Apple, iOS, macOS, design guidelines, CarPlay, interaction patterns,
  visionOS

- [Fluent-Design] Fluent Design System

  Microsoft's design system for Windows, web, and cross-platform applications.
  Defines components, interaction patterns, typography, colour, and motion.
  Foundation for Windows 11, Office, Teams, and Dynamics UI.

  Document: Fluent 2 (2024)\
  URL: <https://fluent2.microsoft.design/>\
  Label: Publication\
  Keywords: Microsoft, Windows, design system, components, typography, motion
  design

- [Material-Design] Material Design

  Google's design system for Android, web, and cross-platform applications.
  Defines components, theming, motion, typography, colour systems, and adaptive
  layouts. The reference design language for Android Automotive and AAOS
  infotainment systems.

  Document: Material Design 3 (2024)\
  URL: <https://m3.material.io/>\
  Label: Publication\
  Keywords: Google, Android, design system, theming, components, Android
  Automotive, AAOS
