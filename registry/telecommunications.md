# Telecommunications

Voice and video coding, optical transport networks, precision time
synchronisation, carrier Ethernet services, IP quality of service, and network
functions virtualisation. Covers ITU-T recommendations, MEF service
specifications, and ETSI NFV/MEC frameworks.

**Contents:** [Standard](#standard) | [Publication](#publication)

## Standard

- [ITU-T-G711] Pulse code modulation (PCM) of voice frequencies

  Defines 64 kbit/s PCM coding for voice telephony. Specifies A-law and mu-law
  companding algorithms. The foundational voice codec for PSTN, VoIP, and
  digital communication systems worldwide.

  Document: ITU-T G.711 (1988, amended 2024)\
  URL: <https://www.itu.int/rec/T-REC-G.711>\
  Label: Standard\
  Keywords: PCM, voice codec, A-law, mu-law, telephony, VoIP, PSTN

- [ITU-T-G722] 7 kHz audio-coding within 64 kbit/s

  Wideband audio codec providing 7 kHz audio bandwidth within a 64 kbit/s
  channel. Used in HD Voice telephony, videoconferencing, and ISDN applications
  delivering improved speech intelligibility over G.711.

  Document: ITU-T G.722 (2012)\
  URL: <https://www.itu.int/rec/T-REC-G.722>\
  Label: Standard\
  Keywords: wideband audio, HD Voice, videoconferencing, ISDN, speech codec

- [ITU-T-G729] Coding of speech at 8 kbit/s using conjugate-structure
  algebraic-code-excited linear prediction (CS-ACELP)

  Low-bitrate speech codec providing toll-quality voice at 8 kbit/s. Widely
  deployed in VoIP gateways and mobile backhaul where bandwidth efficiency is
  critical.

  Document: ITU-T G.729 (2012)\
  URL: <https://www.itu.int/rec/T-REC-G.729>\
  Label: Standard\
  Keywords: speech codec, low-bitrate, CS-ACELP, VoIP, bandwidth efficiency

- [ITU-T-G8275] Precision time protocol telecom profile for phase/time
  synchronization

  Defines IEEE 1588 PTP profiles for telecom networks requiring precise phase
  and time synchronisation. Essential for 5G fronthaul, LTE-TDD, and C-RAN
  deployments where nanosecond-level timing is required.

  Document: ITU-T G.8275 (series)\
  URL: <https://www.itu.int/rec/T-REC-G.8275.1>\
  Label: Standard\
  Keywords: PTP, IEEE 1588, time synchronisation, 5G fronthaul, LTE-TDD, C-RAN,
  telecom

- [ITU-T-G984] Gigabit-capable passive optical networks (GPON)

  Defines gigabit passive optical network technology for fibre-to-the-home and
  fibre-to-the-building access networks. Specifies 2.488 Gbit/s downstream and
  1.244 Gbit/s upstream over single-mode fibre using TDMA.

  Document: ITU-T G.984 (series)\
  URL: <https://www.itu.int/rec/T-REC-G.984.1>\
  Label: Standard\
  Keywords: GPON, FTTH, fibre optics, passive optical network, TDMA, broadband
  access

- [ITU-T-P862] Perceptual evaluation of speech quality (PESQ)

  Objective method for end-to-end speech quality assessment of narrowband
  telephone networks. Compares reference and degraded speech signals to produce
  a quality score. Widely used for VoIP and mobile network testing.

  Document: ITU-T P.862 (2001)\
  URL: <https://www.itu.int/rec/T-REC-P.862>\
  Label: Standard\
  Keywords: speech quality, PESQ, VoIP testing, objective quality, mobile
  network testing

- [ITU-T-Y1541] Network performance objectives for IP-based services

  Defines IP network QoS classes with objectives for delay, jitter, loss, and
  error rate. Specifies six QoS classes from real-time interactive (Class 0) to
  best-effort (Class 5) for end-to-end IP service delivery.

  Document: ITU-T Y.1541 (2011)\
  URL: <https://www.itu.int/rec/T-REC-Y.1541>\
  Label: Standard\
  Keywords: QoS, IP network, delay, jitter, packet loss, service quality,
  real-time

- [MEF-3] Carrier Ethernet services and specifications

  Defines service attributes, architecture, and management for Carrier Ethernet
  services. Covers E-Line, E-LAN, E-Tree, and E-Access services with performance
  objectives for frame delay, jitter, and loss.

  Document: MEF 3.0 Framework (2024)\
  URL: <https://www.mef.net/service-standards/>\
  Label: Standard\
  Keywords: Carrier Ethernet, E-Line, E-LAN, E-Tree, service attributes, MEF

## Publication

- [ETSI-MEC] Multi-access Edge Computing

  ETSI framework for deploying computing resources at the network edge. Defines
  APIs and reference architecture for low-latency applications including
  autonomous vehicles, AR/VR, and IoT at the mobile network edge.

  Document: ETSI GS MEC (series)\
  URL: <https://www.etsi.org/technologies/multi-access-edge-computing>\
  Label: Publication\
  Keywords: edge computing, MEC, low-latency, 5G, IoT, AR/VR, mobile network,
  ETSI

- [ETSI-NFV] Network Functions Virtualisation

  ETSI framework for virtualising network functions on commodity hardware.
  Defines architecture, interfaces, and management for replacing dedicated
  network appliances with software running on standard servers, switches, and
  storage.

  Document: ETSI GS NFV (series)\
  URL: <https://www.etsi.org/technologies/nfv>\
  Label: Publication\
  Keywords: NFV, virtualisation, network functions, commodity hardware, SDN,
  ETSI
