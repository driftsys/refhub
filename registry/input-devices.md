# Input Devices

Standards for the design, ergonomics, and interfaces of physical input devices
and peripherals. Covers keyboard layouts, touch and gesture devices, USB
peripheral interfaces, automotive driver input, and virtual/augmented reality
controllers.

**Contents:** [Standard](#standard)

## Standard

- [ISO-9241-410] Ergonomics of human-system interaction — Design criteria for
  physical input devices

  Defines ergonomic design criteria for physical input devices including
  keyboards, mice, trackballs, joysticks, touchpads, and styli. Covers
  dimensions, force requirements, feedback, and device-specific performance
  criteria.

  Document: ISO 9241-410:2008\
  URL: <https://www.iso.org/standard/38896.html>\
  Label: Standard\
  Keywords: input device ergonomics, keyboards, mice, trackballs, joysticks,
  touchpads, styli, design criteria

- [ISO-9241-420] Ergonomics of human-system interaction — Selection of physical
  input devices

  Provides guidance for selecting appropriate physical input devices based on
  task requirements, user characteristics, and context of use. Covers trade-offs
  between device types for pointing, text entry, and manipulation tasks.

  Document: ISO 9241-420:2011\
  URL: <https://www.iso.org/standard/52938.html>\
  Label: Standard\
  Keywords: input device selection, pointing devices, text entry, user
  characteristics, task requirements

- [ISO-9995] Information technology — Keyboard layouts for text and office
  systems

  Multi-part standard defining keyboard layout principles. Covers physical key
  allocation zones, letter assignment, function key layout, and symbols. Basis
  for national keyboard standards (QWERTY, AZERTY, QWERTZ) and ergonomic
  layouts.

  Document: ISO/IEC 9995 (series, 2009)\
  URL: <https://www.iso.org/standard/51645.html>\
  Label: Standard\
  Keywords: keyboard layout, QWERTY, AZERTY, QWERTZ, key allocation, function
  keys, text entry

- [ISO-15005] Road vehicles — Ergonomic aspects of transport information and
  control systems — Dialogue management principles

  Defines principles for managing driver-vehicle dialogue to minimise
  distraction. Covers interruptibility, resumability, and prioritisation of
  information while driving. See UX/UI domain for full entry.

  Document: ISO 15005:2017\
  URL: <https://www.iso.org/standard/55716.html>\
  Label: Standard\
  Keywords: driver distraction, dialogue management, automotive HMI,
  interruptibility, driver-vehicle interaction

- [OpenXR] OpenXR — Cross-Platform VR/AR Runtime Standard

  Khronos Group API for virtual reality and augmented reality applications.
  Provides a unified interface for head-mounted displays, hand tracking, eye
  tracking, and controller input across VR/AR platforms enabling write-once
  deployment.

  Document: OpenXR 1.1 (Khronos Group, 2024)\
  URL: <https://www.khronos.org/openxr/>\
  Label: Standard\
  Keywords: OpenXR, VR, AR, XR, head-mounted display, hand tracking, eye
  tracking, Khronos Group

- [USB] Universal Serial Bus Specification

  Industry standard for connecting peripherals to host devices. Defines
  electrical, mechanical, and protocol specifications for data transfer and
  power delivery. USB4 provides 120 Gbps throughput and 240W power delivery over
  USB-C connectors.

  Document: USB4 Specification V2.0 (2022)\
  URL: <https://www.usb.org/documents>\
  Label: Standard\
  Keywords: USB, USB4, USB-C, peripheral interface, data transfer, power
  delivery, connectors

- [USB-HID] USB Human Interface Device Class Specification

  Defines the USB device class for human interface devices including keyboards,
  mice, game controllers, and touchscreens. Specifies report descriptors, usage
  tables, and boot protocol enabling plug-and-play HID device recognition across
  all operating systems.

  Document: USB HID 1.11 (2001)\
  URL: <https://www.usb.org/hid>\
  Label: Standard\
  Keywords: USB HID, human interface device, report descriptors, usage tables,
  plug-and-play, game controllers, touchscreens
